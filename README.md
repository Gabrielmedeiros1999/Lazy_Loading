# Lazy_Loading
Projeto de estudo sobre a técnica de Lazy Loading de imagens, desenvolvido utilizando HTML, CSS e JavaScript puro.
A aplicação utiliza uma galeria de imagens de montanhas ao redor do mundo, onde as imagens são inicialmente carregadas em baixa resolução e substituídas por versões em alta qualidade somente quando entram na área visível do usuário.
O objetivo do projeto é demonstrar, conceitos de otimização de performance, redução de carregamento inicial da página e melhoria na experiência do usuário.
